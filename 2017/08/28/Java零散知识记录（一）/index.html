Java零散知识记录（一）

发表于 2017-08-28 | 分类于 杂乱
  1. == 判断两个对象是否是同一个对象，即指针或者引用是否相同。 equals() 判断两个对象的值是否相同
  1. int 强制类型转换会对小数进行取整，例如 (int)3.7 的值为 3 ; Math.round() 会对数字四舍五入，例如 Math.round(3.7)的值为 4；

  2. for循环：for循环有两种形式，第一种为 

    这种包含三个语句： 初始语句，条件语句，控制语句。 第二种为

    float f[]=new float{10];

     for(float x : f ){

}

      包含两个语句：变量，列表，其中列表必须是列表类型，可以是数组、Vector 、List。

  3. break 和 continue：break 用在 while do-while、 for循环中，表示终止当前循环。continue用在while、do-while、for循环中表示跳过本次循环 continue 后面的代码，继续下一次循环。

  4. 数组的初始化： 1>. int [ ] a; 2>.int a [ ]; 3>. int[ ] a={1,2,3,4};

  5. public、private、protected：
     public——>修饰类、变量、方法；其修饰的类具有被其他包中的类访问的可能性，只要在其他包中用import语句引入这个public类，过多的使用public类会使得数据过于暴露，从而降低了安全性
     private ——>修饰变量、方法，只能被自生访问和修改，而且不能被其他任何类（包括其子类）来获取和引用，private修饰符提供了最高级别的保护
     protected——>修饰变量、方法;修饰的成员变量可以被三种类引用：1.该类自身2.与他在同一个包中的其他类3.其他包中该类的子类（任何类都隐式的继承了Object类（基类））

  6. static 和 final: static 命名的变量或使用static{ } 括起来的区域都可以在类被加载时进行初始化，通常用来作为 初始化全局变量 的地方；static 修饰的静态变量必须在定义时就初始化。 final 是最终类和最终方法的修饰符，经常与 static 连用；
     private final ——> 不能被修改的最终变量，编译时产生；
     private static final ——> 不能被修改的静态变量，编译时产生；
     public static final ——> 公共的静态变量
